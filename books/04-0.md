# 公式语法速查

行内公式由一对 $ 符号包裹

```
$a^2 + b^2 = c^2$
```

用 ^ 和 _ 标明上下标,注意上下标的内容（子公式）一般需要**用花括号包裹**，否则上下标只对后面的一个符号起作用

分式使用 \frac{分子}{分母} 来书写

一般的根式使用 \sqrt{...}；表示 n 次方根时写成 \sqrt[n]{...}

乘号 × (\times)、除号 ÷ (\div)、点乘 · (\cdot)、加减号 ± (\pm) / ∓ (\mp)

    \lim_{x \rightarrow 0} 
    \frac{\sin x}{x}=1
\]
